# Go Web 框架

这是一个基于 `net/http` 构建的轻量级 Go Web 框架。

## 功能规划

- [x] **核心功能**
    - [x] 基于 `net/http` 封装核心引擎
    - [x] 提供统一的日志记录功能

- [ ] **路由（Routing）**
    - [ ] 支持静态路由（如 `/about`, `/contact`）
    - [ ] 支持动态路由/参数路由（如 `/users/:id`）
    - [ ] 支持路由分组，可以对一组路由应用相同的前缀或中间件
    - [ ] 支持 RESTful 风格的路由（GET, POST, PUT, DELETE 等）

- [ ] **上下文（Context）**
    - [ ] 封装 `http.ResponseWriter` 和 `*http.Request`
    - [ ] 提供对请求参数、路径参数、表单数据和 JSON 数据的便捷访问方法
    - [ ] 提供快速生成 JSON、HTML、字符串等响应的方法

- [ ] **中间件（Middleware）**
    - [ ] 支持全局中间件
    - [ ] 支持单个路由的中间件
    - [ ] 支持路由分组的中间件
    - [ ] 内置常用的中间件（如 Logger, Recovery）

- [ ] **数据处理**
    - [ ] 支持 JSON 请求体的自动解析和绑定
    - [ ] 支持表单数据的自动解析和绑定

- [ ] **视图/模板**
    - [ ] 支持 HTML 模板渲染
    - [ ] 允许自定义模板函数

- [ ] **静态资源**
    - [ ] 支持配置和提供静态文件服务（CSS, JavaScript, images 等）

- [ ] **错误处理**
    - [ ] 提供统一的 panic 恢复和错误处理机制
    - [ ] 允许用户自定义错误处理函数
